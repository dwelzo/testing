### What are Jest Hooks?

- Jest hooks are lifecycle functions that let you run setup and cleanup code before or after your tests.
- They make your tests clean, isolated, and predictable.

- Set up common test data or environments

- Clean up after tests

- Ensure each test starts with a consistent state

#### Jest Hook List

```
| Hook           | Runs When             | Typical Use                          |
| -------------- | --------------------- | ------------------------------------ |
| `beforeAll()`  | Once before all tests | Set up database connection, mock API |
| `afterAll()`   | Once after all tests  | Close DB connection, clean resources |
| `beforeEach()` | Before each test      | Reset state, render component fresh  |
| `afterEach()`  | After each test       | Cleanup DOM, clear mocks             |

```

for example:

```
// math.test.ts
beforeAll(() => {
  console.log('👉 beforeAll - run once before all tests');
});

afterAll(() => {
  console.log('👋 afterAll - run once after all tests');
});

beforeEach(() => {
  console.log('🔁 beforeEach - runs before every test');
});

afterEach(() => {
  console.log('🧹 afterEach - runs after every test');
});

test('first test', () => {
  console.log('✅ running first test');
  expect(2 + 2).toBe(4);
});

test('second test', () => {
  console.log('✅ running second test');
  expect(3 + 3).toBe(6);
});

```
